## Asentaminen

1. Asenna poetry:

```
poetry install
```

2. Käynnistä sovellus:
```
poetry run invoke start
```


## Testaus ja testikattavuusraportti

Komento testaukselle:

```
poetry run invoke test
```

Komento testikattavuusraportin luonnille:

```
poetry run invoke coverage-report
```
